## 🚀 Quick Start

1. **Install** MenuPilot from WordPress.org (or upload manually)
2. **Navigate** to `MenuPilot → Settings`
3. **Export** a menu: Select menu → Click "Export Menu"
4. **Import** a menu: Upload JSON file → Preview → Import
